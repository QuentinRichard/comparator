# comparator
VueJs Client requests backend to compare 2 web sites.
Projet uses Javascript libraries VueJs2.6 and NestJs

## improvement

### Client
* Add waiting page
* Add toaster (request issues)
* Add Unit test
	
### Server
* Add logger
* Add image comparator
* Add Unit test


## Installation

### Project setup
```
$ npm install
```

### Compiles and minifies for production
```
$ npm run build
```


### Running the app

```bash
$ npm run start
```
